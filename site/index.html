Install

            
Start with the CLI. Embed with Python when ready.

          

          

            

              
CLI quickstart

              
uv pip install socrates-ai
socrates init
export OPENAI_API_KEY=your-key
socrates generate examples/blog_post.yaml --mode guided

            

            

              
Python quickstart

              
from socrates import ContentRequest, Socrates

client = Socrates.from_config()
request = ContentRequest(
    topic="Why onboarding content underperforms",
    audience="B2B SaaS product marketers",
    goal="Generate a publishable blog post",
    platform="blog",
    content_type="blog_post",
)
result = client.generate(request, mode="guided")

FAQ

            
Questions users usually ask before adopting it.

          

          

            

              Is Socrates a general agent framework?
              
No. It is a structured generation layer for content systems, not a multi-agent platform.

            

            

              Does it crawl the web or verify claims?
              
Not in v0.1. The review stage checks publishability signals and anti-patterns, not factual truth.

            

            

              Can I use it with simple rewrite tasks?
              
Yes. Lightweight requests can route to direct mode so you do not pay the multi-stage cost unnecessarily.

            

            

              Why not just write a bigger prompt?
              
Because standards buried in prose are easy for models to blur. Typed frames and outlines are harder to ignore.
